# Fighting-Game
2 player fighting game done in JavaScript, HTML and CSS

Player One controls:
  a - move
  d - move 
  w - jump
  spacebar - attack
  
Player Two controls:
  arrowleft - move
  arrowright - move
  arrowup - jump 
  arrowdown- attack 
  
Game will end when one of the players health value is depleted
To restart just reload the page
